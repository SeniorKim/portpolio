# portpolio
컴퓨터구조 및 운영체제 수강

여기에다가 정리 시작


데이터 : 컴퓨터에서는 0과 1로 구성되어 저장매체에 저장되어 있는 것

명령어 : 컴퓨터에 기설계된 문자로, 이 문자를 입력하면 데이터를 이용해서 무언가를 실행

컴퓨터가 가지고 있는 핵심부품 4가지 : 중앙처리장치, 주기억장치, 보조기억장치, 입ㆍ출력장치

문제 1 : 11111011 (틀림)

문제 2 : 8 6 (틀림)

문제 3 : 0110 1011 (맞음)

컴퓨터는 기본적으로 2진수로 구성되어 2의 n승의 비트로 구성되는데 이 비트를 묶은 기본 단위가 바이트임.
0과 1중 한개의 비트를 저장할 수 있는 플리플롭 여러개로 구성된 레지스트리는 ALU와 제어장치와 같이 CPU를 구성한다. 
CPU는 메모리에 저장된 0과 1로 이루어진 수열을 주소지정방식에 의해 효율적으로 메모리에서 불러와서 저장된 수열(명령어 등)을
실행하고 이 때 버스를 이용한 채널로 효과적으로 처리한다.

0과 1로 구성된 수열의 유형은 숫자이므로, 문자와 같은 유형은 공통된 규칙으로 숫자 패턴이 있는 기준표를 이용하게 되는데 이런 
방식은 아스키코드, 유니코드 등이고 이를 이용해서 숫자가 아닌 문자를 기계가 인식할 수 있는 숫자로도 변환되는데 이러한 숫자로
이루어진 것을 기계어라고 하고 가독성은 기계어에게 좋지 인간에게는 별로 좋지 않다.
