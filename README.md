# portpolio
컴퓨터구조 및 운영체제 수강

여기에다가 정리 시작


데이터 : 컴퓨터에서는 0과 1로 구성되어 저장매체에 저장되어 있는 것

명령어 : 컴퓨터에 기설계된 문자로, 이 문자를 입력하면 데이터를 이용해서 무언가를 실행

컴퓨터가 가지고 있는 핵심부품 4가지 : 중앙처리장치, 주기억장치, 보조기억장치, 입ㆍ출력장치

문제 1 : 11111011 (틀림)

문제 2 : 8 6 (틀림)

문제 3 : 0110 1011 (맞음)

0과 1로 구성된 것이 레지스트리에 있고 이중에서 숫자가 아닌 문자는 아스키코드, 유니코드 등을 이용한 인코딩과 디코딩해서
기계가 인식할 수 있는 기계어를 만듬.
